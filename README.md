## 📖 About

Welcome to the **Cognizant Digital Nurture (DN) 5.0 — Java Cluster Deep Skilling** repository.

This repository contains all the hands-on labs, coding assignments, practical implementations, and mini projects completed during the **7-week Java Full Stack Engineering (React)** deep-skilling program.

The curriculum is designed to prepare engineers for enterprise application development by covering Java, Spring Boot, REST APIs, Hibernate, React, Microservices, DevOps, Docker, Cloud Computing, and Generative AI fundamentals.

# 📖 Program Overview

**Cognizant Digital Nurture 5.0** is an intensive industry-oriented learning initiative designed to bridge the gap between academic knowledge and enterprise software development.

The **Java Full Stack Engineering (React)** learning path equips engineers with modern backend and frontend development skills, enterprise application architecture, cloud-native technologies, DevOps practices, and AI fundamentals.

Throughout the program, participants gain hands-on experience with:

- Java Programming
- Data Structures & Algorithms
- Design Patterns & SOLID Principles
- Oracle PL/SQL
- JUnit 5 & Mockito
- SLF4J Logging
- Spring Core & Maven
- Spring Boot
- Spring Data JPA & Hibernate
- RESTful Web Services
- SonarQube
- Microservices & Spring Cloud
- React.js
- Git & GitHub
- Docker & DevOps
- Cloud Computing
- Generative AI Fundamentals

# 📝 Program Assessments & Evaluation

The Digital Nurture Deep Skilling Program includes continuous evaluation through:

- ✅ Weekly Coding Assignments
- ✅ Hands-on Laboratory Exercises
- ✅ SME Technical Connect Sessions
- ✅ Mentor Reviews
- ✅ Milestone Assessments
- ✅ Coding Challenges
- ✅ Final Comprehensive Assessment

Successful completion of all milestones is a prerequisite for project allocation under the **GenC** and **GenC Next** programs.
